Simple word completion
----------------------
```vim
ctrl + p
```

**or**

```vim
ctrl+n
```

Tag completion
--------------
```vim
ctrl + xctrl + ]
```


Filename completion
-------------------
```vim
ctrl + xctrl + f
```

Context-aware word completion
-----------------------------
```vim
ctrl + x  ctrl + p

**or**

ctrl + x  ctrl + n
```

Context-aware line completion
-----------------------------
```vim
ctrl + x  ctrl + l
```

Language-aware completion
-------------------------
```vim
ctrl + x  ctrl + o
```
